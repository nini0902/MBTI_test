# 「沒用的專業工具」網站

## 專案目標
建立一個看起來很專業但實際上完全沒用的工具集網站。網站採用純前端技術實作，方便部署至 GitHub Pages。

## 功能清單
1. 專業天氣預測
   - 輸出結果只有「會下雨」或「不會下雨」兩種可能
   - 加入專業的氣象圖表動畫（但實際上只是裝飾）
   - 狀態：待開發

2. 今日幸運色
   - 不管按幾次都只會顯示灰色
   - 附上偽專業的色彩分析報告
   - 狀態：待開發

## 技術規格
- 純前端實作（HTML, CSS, JavaScript）
- 使用 CDN 載入所需套件
- 計畫使用的套件：
  - Chart.js（用於顯示無意義但看起來很專業的圖表）
  - Anime.js（用於製作華麗的動畫效果）
  - Bootstrap（基本排版）
